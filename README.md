## ✨ What is Atheno?
Atheno is a **modern, intelligent learning platform** that transforms how students manage and master their studies.

Instead of switching between multiple apps for courses, to-dos, flashcards, timers and progress tracking —  
Atheno brings everything together in one **beautiful AI-powered dashboard**.

---

## 🧠 Core Features

| Category | Features |
|---------|----------|
| 📚 Learning | Course Manager • AI Roadmaps • Notes & Materials |
| 🧠 Memory | AI Flashcards • Recall Questions • Spaced Repetition |
| ⏱ Focus | Pomodoro Timer • Study Sessions History |
| 📝 Productivity | Task Manager • Deadlines • Reminders |
| 📈 Analytics | Focus Time Graphs • Streaks • Progress Insights |
| 🔐 Authentication | Email/Password Login • Google OAuth |
| 💎 User Experience | Premium Glass UI • Animations • Fully Responsive |
